# Configuração de DHCP em Roteador Wireless

Este projeto faz parte do meu portfólio de estudos em redes e suporte de TI.

Neste laboratório utilizei o Cisco Packet Tracer para simular a configuração de uma rede local com um roteador wireless configurado como servidor DHCP.

## Objetivo

Praticar conceitos fundamentais de redes:

- configuração de roteador
- funcionamento do DHCP
- atribuição automática de endereços IP
- testes de conectividade em rede local

## Ferramentas utilizadas

- Cisco Packet Tracer
- Simulação de rede LAN
- Configuração de DHCP
- Testes de conectividade com ping
